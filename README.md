# 🎭 Шпион — Telegram Bot

Telegram-бот для групповой игры «Шпион» на **aiogram 3.x** с SQLite.

## 🎮 Особенности

- 📂 **6 категорий персонажей** — кино, игры, книги, музыканты, история, города
- 🎯 **2 режима игры** — классические описания или вопросы да/нет
- 🤡 **Роль «Провокатор»** — мирный с фейковым персонажем, путает шпионов
- 👤 **Режим ведущего** — выбери игрока, который загадает персонажа и не будет играть
- 💡 **Подсказки для шпиона** — первая буква, категория или длина имени (1 раз за игру)
- 🕵️ **От 1 до 10 шпионов** — настраиваемое количество
- 🔄 **Перевыбор персонажа** — голосование за смену загаданного слова
- 💌 **Тайные письма** — отправляй сообщения другим игрокам
- 📊 **Статистика** — отслеживай свои победы и поражения
- ✨ **Кастомные персонажи** — добавляй своих через команды

## 📜 Правила

### Роли

| Роль | Знает персонажа? | Цель |
|------|------------------|------|
| 👤 **Мирный житель** | ✅ Да | Вычислить шпионов |
| 🕵️ **Шпион** | ❌ Нет | Угадать персонажа или не быть раскрытым |
| 🤡 **Провокатор** | ⚠️ Знает ФЕЙКОВОГО | Запутать мирных, победить со шпионами |
| 👤 **Ведущий** | ✅ Да | Загадать персонажа, не играть |

### Как играть

1. Добавьте бота в группу и дайте права на чтение сообщений
2. В группе отправьте `/spy` — создастся лобби
3. Игроки нажимают **🎮 Присоединиться**
4. Создатель настраивает игру через **⚙️ Настроить игру**:
   - Количество шпионов (1-10)
   - Категорию персонажей
   - Режим игры (описания / вопросы)
   - Провокатор (вкл/выкл)
   - Ведущий (выбрать из игроков)
5. Создатель нажимает **🚀 Начать игру**
6. Бот пишет каждому игроку в ЛС его роль

### Режим «Классика» (описания)

- Мирные жители по очереди называют **1 признак** персонажа
- Нельзя называть имя напрямую!
- После всех описаний — **обсуждение ГОЛОСОМ**
- В чате **не пишите подсказки** — шпионы читают!

### Режим «Вопросы»

- Игроки по очереди задают друг другу вопросы о персонаже
- Вопросы должны быть на **да/нет**
- Шпион должен отвечать, не зная ответа — это сложно!
- 2 раунда вопросов, затем обсуждение

### Победа

- 🕵️ **Шпион** может в любой момент написать боту в ЛС: `/guess ИмяПерсонажа`
- 👥 **Мирные** голосуют командой `/vote` в группе

## 💡 Подсказки для шпиона

Шпион может запросить **1 подсказку за игру** командой `/hint` в ЛС:

- 🔤 **Первая буква** персонажа
- 📏 **Длина имени** в символах

---

## 🤡 Провокатор

Особая роль для опытных игроков:

- Получает **фейкового персонажа** (например, «Кот Шрёдингера»)
- Описывает его как настоящего
- Путает мирных своими описаниями
- Побеждает вместе со шпионами

---

## 👤 Ведущий

- Выбирается создателем из списка игроков
- Получает персонажа в ЛС и может **подтвердить** или **запросить другого**
- Может задать **своего персонажа** командой `/setchar Имя`
- **Не участвует в игре** — не получает роль, не голосует

---

## ⚠️ Важно

- Перед игрой **каждый игрок должен написать боту `/start` в ЛС**
- У бота должен быть доступ к сообщениям группы (или отключен Privacy Mode)

## 🚀 Установка и запуск

### Через Python

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте бота через [@BotFather](https://t.me/BotFather), получите токен

3. Создайте файл `.env`:
```
BOT_TOKEN=your_bot_token_here
ADMIN_USERNAME=your_telegram_username
```

4. Запустите бота:
```bash
python -m bot
```

Или двойным кликом по `start.bat` (Windows).

### Через Docker

```bash
docker-compose up -d
```

## 📝 Добавление персонажей

Откройте файл `data/characters.json` и добавьте персонажей в нужную категорию:

```json
{
  "categories": {
    "movies": {
      "name": "Кино и сериалы",
      "emoji": "🎬",
      "characters": ["Персонаж 1", "Персонаж 2"]
    }
  }
}
```

Изменения применятся сразу, перезапуск не требуется.

## 📋 Команды

### В группе
| Команда | Описание |
|---------|----------|
| `/spy` | Создать лобби |
| `/join` | Присоединиться к игре |
| `/vote` | Начать голосование |
| `/stop` | Остановить игру (только создатель) |
| `/status` | Статус текущей игры |
| `/players` | Список игроков |
| `/kick` | Исключить игрока (ответом на сообщение) |
| `/reroll` | Голосование за смену персонажа |
| `/help` | Помощь и правила |

### В ЛС бота
| Команда | Описание |
|---------|----------|
| `/start` | Активировать бота |
| `/guess Имя` | Угадать персонажа (для шпиона) |
| `/hint` | Получить подсказку (для шпиона, 1 раз) |
| `/send @username Текст` | Отправить тайное письмо |
| `/setchar Имя` | Установить персонажа (для ведущего) |
| `/mystats` | Моя статистика |
| `/help` | Помощь |
