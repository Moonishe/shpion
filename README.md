## 🎮 Режимы игры

| Режим | Описание |
|-------|----------|
| 📝 **Классика** | Игроки описывают персонажа вслух, 1 признак за ход |
| ❓ **Вопросы** | Игроки задают да/нет вопросы друг другу по кругу |
| 🎭 **Слепой шпион** | Шпион получает фейкового персонажа и думает что мирный |
| 👤 **Все мирные** | 5% шанс — шпионов нет, все знают слово |
| 🕵️ **Все шпионы** | 5% шанс — все шпионы, хаос на голосовании |
| 🔀 **Split-слова** | 5% шанс (>5 игроков) — мирные получают разные слова |

---

## 👥 Роли

| Роль | Знает слово | Цель |
|------|------------|------|
| 👤 **Мирный** | ✅ | Вычислить шпиона |
| 🕵️ **Шпион** | ❌ | Угадать слово или выжить |
| 🤡 **Провокатор** | ⚠️ фейк | Путать мирных, играет за шпионов |
| 🔀 **Путаник** | ⚠️ другое | Другой персонаж из категории |
| 👑 **Ведущий** | ✅ | Загадывает слово, не играет |

### Взвешенные роли
Шанс стать шпионом/провокатором зависит от истории: был шпионом → шанс ↓ в 2 раза. Не был → восстанавливается. Никогда не ноль.

---

## 💡 Подсказки (автоматические)

4 типа: 🎲 случ. буква, 🔤 первая, 🔤 последняя, 📏 длина.

| Шпионов | Частота |
|---------|---------|
| 1 | Каждый раунд со 2-го |
| 2 | Каждые 2 раунда, разные |
| N | Каждые N раундов |

Мирные получают: «🔔 Шпионы получили подсказку.»
При выбывании провокатора — всем шпионам бонус-подсказка.

---

## 🗳️ Голосование

- `/vote` из фазы обсуждения. Одно за раз.
- Счётчики голосов в реальном времени: `🎯 Иван (3)`
- ❌ Отмена голосования большинством
- >75% участия для раскрытия результата
- Авто-победа: шпионы ≥ мирных → игра окончена
- Fuzzy matching для `/guess`: Левенштейн, порог 45%

## 📜 Как играть

1. Добавьте бота в группу
2. Каждый игрок пишет `/start` боту в ЛС
3. В группе `/spy` → лобби
4. Игроки жмут **🎮 Присоединиться**
5. Создатель настраивает: шпионы, категория, режим, провокатор, ведущий
6. **🚀 Начать игру** — бот проверит что все активированы
7. Роли приходят в ЛС. Игра началась!

## 📋 Команды

### В группе

| Команда | Кто | Описание |
|---------|-----|----------|
| `/spy` | Все | Создать лобби |
| `/join` | Все | Присоединиться |
| `/vote` | Все | Начать голосование |
| `/stop` | Создатель/Админ | Остановить |
| `/kick` | Создатель/Админ | Кикнуть (реплай) |
| `/reroll` | Все | Голосование за смену слова |
| `/status` | Все | Статус игры |
| `/players` | Все | Список игроков |
| `/help` | Все | Помощь |

### В ЛС бота

| Команда | Кто | Описание |
|---------|-----|----------|
| `/start` | Все | Активировать бота |
| `/guess Имя` | Шпион | Угадать слово |
| `/hint` | Шпион | «Приходят автоматически» |
| `/send @user Текст` | Все | Тайное письмо (1 раз) |
| `/setchar Имя` | Ведущий | Свой персонаж |
| `/mystats` | Все | Статистика |

## ⚙️ Админ-контроль

Создатель игры ИЛИ админ бота (`ADMIN_USERNAME` в `.env`) могут:
- Менять настройки
- Запускать/останавливать игру
- Кикать игроков на любом этапе

## 🚀 Установка

```bash
pip install -r requirements.txt
```

`.env`:
```
BOT_TOKEN=your_token
ADMIN_USERNAME=your_username
```

```bash
python -m bot
```

Windows: двойной клик по `start.bat`.
