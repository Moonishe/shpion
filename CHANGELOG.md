# Changelog

## v1.1 — Май 2026

### 🆕 Новые фичи

| Фича | Описание |
|------|----------|
| **🎭 BLIND\_SPY** | Слепой шпион — получает фейк-персонаж, думает что мирный, `/guess` и `/hint` заблокированы |
| **🔀 Split-слова (5%)** | >5 игроков — половина мирных получает слово А, половина Б. Шпион угадывает любое. Никто не знает |
| **⚖️ Взвешенные роли** | Был шпионом → шанс ↓ в 2 раза. Не был → восстанавливается. Никогда не ноль |
| **💡 Авто-подсказки** | 4 типа, N шпионов → каждые N раундов. Мирным: «🔔 Шпионы получили подсказку» |
| **🔢 Fuzzy matching** | Левенштейн, порог 45%. «Гарри Потный» → «Гарри Поттер» ✅ |
| **🚪 Полное выбывание** | При ошибочном голосовании игрок удаляется из игры |
| **👢 Кик всегда** | Создатель/админ может кикать на любом этапе |
| **👑 Админ-контроль** | `ADMIN_USERNAME` управляет любой игрой в любом чате |
| **🛡️ Проверка /start** | Игроки не написавшие боту в ЛС блокируют запуск |
| **🏆 Авто-победа** | Шпионы ≥ мирных → игра кончается. 1v1 тупик разрешается |
| **🗳️ Счётчики голосов** | `🎯 Иван 👑 — 3` в реальном времени |
| **❌ Отмена голосования** | Большинство → сброс |
| **⏱️ Пауза 1 мин** | После раунда, вместо 10 сек |
| **📂 Clash Royale** | 120 карт в отдельной категории |
| **👑 Корона** | Создатель лобби с 👑 во всех списках |
| **📋 /version** | Версия бота и ссылка на GitHub |

### 🔧 Техническое

- `_can_control()` — единая проверка прав (создатель + админ)
- `normalize_for_comparison` — цифры↔слова, транслит, ё→е
- `_levenshtein` — O(n) память, Unicode
- `_weighted_choice_idx` — взвешенный выбор без повторений
- `get_streaks` / `update_streaks` — история ролей в SQLite
- Миграции: `spy_streak`, `provocateur_streak`, `split_character`
- `threading.Lock` на `characters.json`
- `_started_users` — отслеживание активированных игроков
- Сентинел `"*"` вместо `"all"` для «все категории»
